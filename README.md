# 💧 Controle de Níveis de Água

Este projeto foi desenvolvido em Python como parte de uma atividade prática.

## Objetivo

Simular o monitoramento de um reservatório de água, exibindo mensagens no terminal com cores diferentes de acordo com o nível.

## Tecnologias utilizadas

- Python
- Biblioteca Colorama

## Funcionalidades

- Exibição de 5 níveis do reservatório
- Uso de cores para indicar a situação:
  - Vermelho → Muito baixo (crítico)
  - Amarelo → Baixo
  - Verde → Médio
  - Ciano → Alto
  - Azul → Muito alto (alerta)

## Estrutura do código

O sistema utiliza:

- Lista para armazenar os níveis
- Função para exibir mensagens coloridas
- Loop para simulação do monitoramento
